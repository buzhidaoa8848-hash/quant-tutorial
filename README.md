# quant-tutorial

> 量化投资学习项目 — 从零搭建你的第一个量化策略

## 目标

学习 Python 量化的完整流程：数据获取 → 策略开发 → 回测 → 分析

## 路线图

- [x] 项目框架搭建

- [ ] ETF 动量轮动策略
- [ ] backtrader 回测
- [ ] 绩效分析
- [ ] 更多策略...

```
quant-tutorial/
```
